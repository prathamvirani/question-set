Engineering Mathematics: Functions Test

Questions & Detailed Answers

Questions üìù

1. Basic Definitions

In your own words, what is the mathematical definition of a function? Why is the "vertical line test" a useful tool for identifying functions from a graph?

2. Domain and Range

Determine the domain and range for the function $y = \sqrt{x^2 - 9}$. Express your answers in interval notation.

3. Identifying Functions

Which of the following equations represents a function? Justify your answer for each.
a) $y^2 = 3x + 1$
b) $y = |x - 4|$
c) $x^2 + y^2 = 25$

4. Rational Functions

Given the function $f(x) = \frac{2x+1}{x-3}$, find the domain and range.

5. Evaluating Functions

If $f(x) = 3x^2 - 2x + 5$, evaluate $f(a+1)$.

6. Domain of Radical Functions

Determine the domain of the function $y = \frac{1}{\sqrt{4-x}}$.

7. Range with Restricted Domain

A function is defined as $f(x) = x^3$ for the domain $-3 \le x \le 3$. What is the range of this function?

8. Arithmetic Operations

Given $f(x) = 3x + 2$ and $g(x) = x^2 - 1$, find:
a) $(f+g)(x)$
b) $(fg)(x)$
c) $(\frac{f}{g})(x)$ and state its domain.

9. Inverse Functions

If $f(x) = 2x - 5$, find the inverse function, $f^{-1}(x)$.

10. Composite Functions

Given $f(x) = \sqrt{x+1}$ and $g(x) = x^2$, find:
a) $(f \circ g)(x)$
b) $(g \circ f)(x)$

11. Inverse of Rational Functions

Find the inverse of the function $f(x) = \frac{2x+3}{x-1}$.

12. Nested Composite Functions

Given $a(x) = 2x$, $b(x) = x^2$, $c(x) = x+3$, and $d(x) = \sqrt{x}$, find $f(x) = a[b(c[d(x)])]$.

13. Decomposing Functions

If $h(x) = (2x+1)^3$, find two functions $f(x)$ and $g(x)$ such that $h(x) = (f \circ g)(x)$.

14. Verifying Inverses

Given $f(x) = 5x-3$ and $g(x) = \frac{x+3}{5}$, show that $f(x)$ and $g(x)$ are inverses of each other.

15. Linear Cost Function

A factory's production cost is modeled by a linear function. The fixed cost is RM 12,000, and the variable cost per unit is RM 85. Write a function $C(x)$ that represents the total cost of producing $x$ units.

16. Analyzing Quadratic Functions

For the quadratic function $f(x) = -2x^2 + 8x - 6$, find:
a) The vertex.
b) The axis of symmetry.
c) The y-intercept.
d) The x-intercepts.

17. Projectile Motion

The height (in meters) of a rocket launched from the ground is given by the function $h(t) = -4.9t^2 + 196t$, where $t$ is the time in seconds. What is the maximum height the rocket reaches, and how long does it take to get there?

18. Graphing Quadratic Functions

Sketch the graph of the function $y = (x+3)^2 - 4$. Label the vertex and any intercepts.

19. Maximizing Profit

The daily profit, $P(x)$, from the sale of $x$ units of a product is given by $P(x) = -0.01x^2 + 40x - 15000$. How many units must be sold to maximize the daily profit? What is the maximum profit?

20. Equation of a Parabola

Find the equation of the parabola that has a vertex at $(1, -2)$ and passes through the point $(3, 6)$.

21. Identifying Polynomials

Which of the following are polynomial functions? For those that are, state the degree.
a) $f(x) = 5x^4 - \frac{2}{x} + 1$
b) $g(t) = \sqrt{3}t^3 - 2t^2 + 7$
c) $h(x) = 9 - x^6$

22. Chemical Engineering Application

The concentration of a substance is given by $C(t) = C_0 e^{-kt}$, where $C_0 = 5$ mol/L and $k = 0.25 \text{ hr}^{-1}$.
a) What is the concentration after 4 hours?
b) How long until the concentration is 1 mol/L?

23. Electrical Engineering Application

Power is $P(V) = \frac{V^2}{R}$ with $R=50 \Omega$. Voltage is $V(t) = 120e^{-0.1t}$.
a) Find the composite function $P(V(t))$.
b) What is the power at $t=10$ seconds?

24. Mechanical Engineering Application

A component cools according to $T(t) = T_{env} + (T_0 - T_{env})e^{-kt}$. Given $T_0 = 150^\circ\text{C}$, $T_{env} = 20^\circ\text{C}$, and $k = 0.05 \text{ min}^{-1}$.
a) Find the temperature after 30 minutes.
b) How long to cool to $30^\circ\text{C}$?

25. Civil Engineering Application

Beam deflection is $D(x) = 0.0005x^4 - 0.01x^3 + 0.05x^2$. What is the degree of this polynomial, and what is the physical meaning of $D(0)$?

1. Solution

A function is a rule assigning each input value exactly one unique output value. The vertical line test is a graphical check; if any vertical line crosses the graph more than once, it fails the test because one input (x-value) corresponds to multiple outputs (y-values).

2. Solution

For the domain, we need $x^2 - 9 \ge 0 \implies x^2 \ge 9$. This holds for $x \le -3$ or $x \ge 3$.
Domain: $(-\infty, -3] \cup [3, \infty)$

The square root function produces non-negative values.
Range: $[0, \infty)$

c) Not a function. This is the equation of a circle, which fails the vertical line test.

4. Solution

Range: The function represents a hyperbola with a horizontal asymptote at $y=2$. Range is $(-\infty, 2) \cup (2, \infty)$.

7. Solution

8. Solution

a) $(f+g)(x) = (3x+2) + (x^2-1) = \mathbf{x^2 + 3x + 1}$

b) $(fg)(x) = (3x+2)(x^2-1) = \mathbf{3x^3 + 2x^2 - 3x - 2}$

c) $(\frac{f}{g})(x) = \mathbf{\frac{3x+2}{x^2-1}}$. Domain requires $x^2-1 \ne 0$, so $x \ne \pm 1$. Domain: $(-\infty, -1) \cup (-1, 1) \cup (1, \infty)$

9. Solution

$\mathbf{f^{-1}(x) = \frac{x+5}{2}}$

10. Solution

a) $(f \circ g)(x) = f(g(x)) = f(x^2) = \mathbf{\sqrt{x^2+1}}$

b) $(g \circ f)(x) = g(f(x)) = g(\sqrt{x+1}) = (\sqrt{x+1})^2 = \mathbf{x+1}$

11. Solution

Let $y = \frac{2x+3}{x-1}$. Swap variables: $x = \frac{2y+3}{y-1}$.

$x(y-1) = 2y+3 \implies xy-x = 2y+3 \implies xy-2y = x+3$

$y(x-2) = x+3 \implies \mathbf{f^{-1}(x) = \frac{x+3}{x-2}}$

14. Solution

Since both compositions equal $x$, they are inverses.
