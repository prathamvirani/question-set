Rules of Indices

$a^m \times a^n = a^{m+n}$
$a^m \div a^n = a^{m-n}$
$(a^m)^n = a^{mn}$
$a^0 = 1$
$a^{-m} = \frac{1}{a^m}$
$a^{\frac{1}{m}} = \sqrt[m]{a}$
$a^{\frac{n}{m}} = \sqrt[m]{a^n}$

Rules of Logarithms

$\log_a(x \times y) = \log_a x + \log_a y$
$\log_a(x \div y) = \log_a x - \log_a y$
$\log_a(x^n) = n \log_a x$
Change of Base: $\log_a x = \frac{\log_b x}{\log_b a}$

Quadratic Formula

For $ax^2 + bx + c = 0$
$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$

Trigonometric Identities

$\cos^2 x + \sin^2 x = 1$
$\sec^2 x = 1 + \tan^2 x$
$\csc^2 x = 1 + \cot^2 x$
$\sin(2x) = 2 \sin x \cos x$
$\cos(2x) = \cos^2 x - \sin^2 x$

Hyperbolic Functions

$\sinh x = \frac{e^x - e^{-x}}{2}$
$\cosh x = \frac{e^x + e^{-x}}{2}$
$\tanh x = \frac{\sinh x}{\cosh x}$

Inverse Hyperbolic Functions

$\sinh^{-1} x = \ln(x + \sqrt{x^2 + 1})$
$\cosh^{-1} x = \ln(x + \sqrt{x^2 - 1})$
$\tanh^{-1} x = \frac{1}{2} \ln(\frac{1+x}{1-x})$

Basic Definitions

$j = \sqrt{-1} \implies j^2 = -1$
Modulus: $|z| = \sqrt{a^2 + b^2}$
Argument: $\arg(z) = \tan^{-1}(\frac{b}{a})$
Polar Form: $z = r(\cos \theta + j \sin \theta)$

De Moivre's Theorem

$z^n = r^n (\cos n\theta + j \sin n\theta)$
$\sqrt[n]{z} = \sqrt[n]{r} (\cos\frac{\theta + 2\pi k}{n} + j \sin\frac{\theta + 2\pi k}{n})$

Expansions

$z + \frac{1}{z} = 2 \cos \theta$
$z - \frac{1}{z} = j2 \sin \theta$
$z^n + \frac{1}{z^n} = 2 \cos n\theta$
$z^n - \frac{1}{z^n} = j2 \sin n\theta$

Arithmetic Progression

nth term: $a_n = a_1 + (n-1)d$
Sum: $S_n = \frac{n}{2}[2a + (n-1)d]$

Geometric Progression

nth term: $a_n = a_1 r^{n-1}$
Sum: $S_n = \frac{a(1-r^n)}{1-r}$ for $r<1$
Sum to infinity: $S_\infty = \frac{a}{1-r}$

Common Power Series

$e^x = 1 + x + \frac{x^2}{2!} + \dots$
$\sin x = x - \frac{x^3}{3!} + \frac{x^5}{5!} - \dots$
$\cos x = 1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \dots$

Vector Magnitude

2D: $|\mathbf{a}| = \sqrt{a_1^2 + a_2^2}$
3D: $|\mathbf{a}| = \sqrt{a_1^2 + a_2^2 + a_3^2}$
Unit Vector: $\hat{\mathbf{r}} = \frac{\mathbf{r}}{|\mathbf{r}|}$

Scalar (Dot) Product

$\mathbf{a} \cdot \mathbf{b} = a_1 b_1 + a_2 b_2 + a_3 b_3$
$\mathbf{a} \cdot \mathbf{b} = |\mathbf{a}||\mathbf{b}| \cos \theta$
If $\mathbf{a} \cdot \mathbf{b} = 0$, vectors are orthogonal.

Vector (Cross) Product

$\mathbf{a} \times \mathbf{b} = \begin{vmatrix} \mathbf{i} & \mathbf{j} & \mathbf{k} \\ a_1 & a_2 & a_3 \\ b_1 & b_2 & b_3 \end{vmatrix}$
$|\mathbf{a} \times \mathbf{b}| = |\mathbf{a}||\mathbf{b}| \sin \theta$
