2. Kata Adjektif / Adjectives

These adjectives are used to describe a person's physical features and personality traits.

Physical Appearance

  • tinggi/rendah: tall/short
  • panjang/pendek: long/short
  • gemuk/kurus: fat/thin
  • besar/kecil: big/small
  • lurus/kerinting: straight/curly (hair)

Perwatakan / Characteristics

  • baik: kind
  • sopan: polite
  • penyabar: patient
  • pemurah: generous
  • peramah: friendly
  • pemalas: lazy
  • pemarah: hot-tempered

3. Grammar & Antonyms

Grammar: Basic Sentence Pattern (FN + FA)

To describe someone, use the **Noun Phrase + Adjective Phrase** pattern. It's simple: **Subject + Adjective**.

`Rambut dia pendek.` (His/Her hair is short.)

`Ahmad tinggi.` (Ahmad is tall.)

`Mata kanan awak bengkak.` (Your right eye is swollen.)
